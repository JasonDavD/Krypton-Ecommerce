A compact button for a single icon (cart, search, favorite, carousel arrows). Uses Lucide names; call `lucide.createIcons()` after render.

```jsx
<IconButton icon="shopping-cart" label="Carrito" variant="soft" />
<IconButton icon="heart" label="Favorito" variant="outline" round />
<IconButton icon="chevron-right" label="Siguiente" variant="solid" round />
```

Variants: `solid` (blue) · `cta` (orange) · `soft` (blue tint, default) · `outline` · `ghost`. Sizes `sm|md|lg`, `round` for circular.
