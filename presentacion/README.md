# Presentación — Krypton E-commerce (EFSRT VI)

Material para la exposición del proyecto. Todo autocontenido.

## Contenido

```
presentacion/
├── index.html              ← el deck de diapositivas (abrir con doble clic)
├── assets/                 ← fuentes Kanit + logos (marca Krypton)
└── guiones/                ← guion de cada expositor (1 archivo c/u)
    ├── guion-1-naomi-caso-de-negocio.md
    ├── guion-2-billy-arquitectura.md
    ├── guion-3-jason-backend.md
    ├── guion-4-edward-frontend.md
    └── guion-5-luis-nube-cierre.md
```

## Cómo presentar

1. Abrí **`index.html`** en el navegador (doble clic). Poné pantalla completa con **F11**.
2. Navegá las diapositivas:
   - **→** / **Espacio** / **PageDown** = siguiente
   - **←** / **PageUp** = anterior
   - **Home** / **End** = primera / última
3. Cada quien sigue su **guion** (en `guiones/`), que tiene el texto a decir, las señas de cada slide, el pase al siguiente expositor y preguntas frecuentes.
